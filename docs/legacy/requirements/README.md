# Legacy Requirements

Purpose:

- preserve business context,
- keep historical ideas,
- and avoid losing algorithm rationale.

Use these docs as reference, not as the implementation source of truth.

Current implementation source of truth:

- `docs/architecture/grand_design.md`
- `docs/architecture/vue_implementation_plan.md`
- `docs/architecture/implementation_roadmap.md`

Contents:

- `algorithm/`: original metric, parser, validation, and spreadsheet-based requirement docs
- `architecture/`: earlier backend, frontend, and infra notes that predate the final Vue-first architecture
